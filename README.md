## DESKRIPSI PROJECT

Project ini merupakan implementasi website untuk Ujian Tengah Semester — Pemrograman Web 1.
Website ini bertema Kantin Bu Vida dengan desain dark purple aesthetic dan terdiri dari 3 halaman utama:

### 1. HALAMAN LOGIN (index.html)

•Form login dengan input email dan password (password = NIM).

•Validasi menggunakan JavaScript.

•Jika berhasil login → redirect ke dashboard.html.

•Terdapat desain modern dengan efek animasi halus.

### 2. HALAMAN DASHBOARD (dashboard.html)

•Menampilkan ringkasan data dalam bentuk 3 card:

  •Total Produk
  
  •Total Penjualan
  
  •Total Revenue

•Terdapat sidebar navigasi (Dashboard, Data Produk, Logout).

•Tombol untuk menuju halaman Data Produk.

### 3. HALAMAN DATA PRODUK (products.html)

•Menampilkan daftar produk dalam bentuk tabel.

•Data produk berasal dari array JavaScript.

•Memiliki fitur:

  •Edit → menampilkan alert.
  
  •Delete → menghapus data dengan konfirmasi.

•Desain tabel modern, aesthetic, dan responsif.
